# Airline Reservation System (Java + JDBC)

##  Project Overview
This is a console-based Airline Reservation System developed using **Java**, **JDBC**, and **MySQL**.
It allows users to register, login, search flights, and book tickets.

##  Technologies Used
- Java (JDK 24)
- JDBC
- MySQL
- Eclipse IDE
- Git & GitHub

##  Features
- User Registration & Login
- Search Flights
- Book Tickets
- Database connectivity using JDBC
- DAO Design Pattern

##  Project Structure
- `com.airline.dao` → Database operations
- `com.airline.model` → Entity classes
- `com.airline.test` → Test / Main classes

##  How to Run
1. Import project into Eclipse
2. Configure MySQL database
3. Update DB credentials in `DBConnection.java`
4. Run test classes from `com.airline.test`
